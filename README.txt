The stopwatch module allows you to record the time on an activity, such as reading a text, and link it to a evaluable activity as a quiz. 

--------------------------------------------------------------------------------
To Install:

1. Load the stopwatch module directory into your "mod" subdirectory.
2. Visit your admin page to create all of the necessary data tables.

--------------------------------------------------------------------------------
To Upgrade:

1. Copy all of the files into your 'mod/stopwatch' directory.
2. Visit your admin page. The database will be updated.

--------------------------------------------------------------------------------
To Uninstallation:

1. Into admin, plugin, plugins overview, link to uninstall
2. In '/mod/' remove the folder 'stopwatch'.
